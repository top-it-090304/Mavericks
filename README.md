# Mavericks
Концепция (О чем игра)
Это баскетбольная аркада на один палец. Никаких правил, матчей и соперников. Только ты, мяч и кольцо. Задача — кидать и не промахиваться. Ошибся — игра заканчивается.

Механика (Как играется)

Ты тянешь мяч пальцем назад — появляется пунктирная линия.

Отпускаешь — мяч летит.

Кольцо каждый раз двигается на новое место.

Одно касание — один бросок.

Фишки, которые затягивают

Идеальные попадания. Если мяч залетает, не коснувшись кольца, ты получаешь больше очков. За серию таких бросков множитель растет до х10.

Рикошеты. Ударил мячом о стену или щит, а он залетел в корзину — очки удваиваются.

Красивый мусор. Ты открываешь новые мячи: баскетбольный, теннисный, смайлик, светящийся, неваляшка. Они не дают преимущества, их просто приятно коллекционировать.
